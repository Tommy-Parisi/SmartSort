### Frequently Asked Questions

**Is it really private? Does it send my files anywhere?**
Yes, it's really private. Smart Sort runs entirely on your Mac using a local AI model. Your file contents never leave your machine — no cloud uploads, no telemetry, no API calls. The folder naming model runs locally too.

**What about safety? Will it move files I didn't want moved?**
No files move without your confirmation. Every sort shows you a preview of what will happen before anything changes. And if you don't like the result, Undo rolls every file back to exactly where it started.

**What does the trial actually let me do?**
Everything — up to 100 files per sort. Full semantic clustering, AI naming, preview editor. Enough to sort a real folder and feel the difference. Most Downloads folders have 500–2,000 files, so the trial shows you what it can do without finishing the job.

**Does it work on Windows?**
Not yet. macOS only for now. Windows support is in progress — if you want early access when it ships, email us and we'll let you know.

**What file types does it understand?**
PDFs, Word documents, plain text, code files (Python, JS, Rust, and more), images (via OCR), spreadsheets, PowerPoint, JSON, and archives. If it can read it, it can sort it.
